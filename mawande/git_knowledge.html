Git Knowledge
  • type:sudo apt-get install git (To install git)
  • type:git clone [https://github.com/mawande/codeX_OnlineSetup] (to clone files from github to computer)
  • type:git config --global user.email "mawande@projectcodex.co"( so that system can recognize as the owner of the branch)
  • type: git add projects (to add the file)
  • type:git status (to check the status of the file, whether its untracked or not)
  • type:touch .gitignore(to not push up copies)
  • type: git commit -m 'my website'( to commit and add a message)
  • type:git push origin master( to upload online to github)
